Introduction

When Javascript came out I moved to a UI that would help me hack into java so that I could make it do what I wanted. Eventually I transitioned into blogging. Before I was done blogging I had five blogs that linked to eachother seemlessly with a complex menu system. I am in the coding boot camp because I am eager to take this hobby of mine and turn it into a career.
